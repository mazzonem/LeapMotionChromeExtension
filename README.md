LeapMotionChromeExtension
=========================

Leap Motion Chrome Extension - Multi OS

Use
=========================
Add leap.min.js and enjoy :)

Installation
http://developer.chrome.com/extensions/getstarted.html#unpacked


Actions
=========================
Scroll : 5 fingers

Swipe Left or Right : Previous and Forward

Circle : Refresh

Click : 2 fingers
